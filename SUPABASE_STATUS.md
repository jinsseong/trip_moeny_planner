## 서버 메시지가 안 보이는 이유

"✅ Supabase 연결 모드로 실행됩니다." 메시지가 안 보이는 경우:

### 1. 서버가 재시작되지 않음
환경변수를 추가한 후 서버를 재시작해야 합니다.

**해결 방법:**
```bash
# 현재 실행 중인 서버 종료 (Ctrl+C)
# 그 다음 다시 시작:
npm run dev
# 또는
node server.js
```

### 2. Vite 개발 서버 사용 중
`npm run dev`로 실행하면 Vite 개발 서버가 실행되고, Express 서버(`server.js`)는 별도로 실행해야 합니다.

**해결 방법:**
```bash
# 터미널 1: Vite 개발 서버
npm run dev

# 터미널 2: Express 서버
node server.js
```

### 3. 메시지 확인 방법
서버를 시작하면 터미널에 다음 중 하나가 표시됩니다:

- ✅ `✅ Supabase 연결 모드로 실행됩니다.` → 정상
- ⚠️ `⚠️ JSON 파일 모드로 실행됩니다.` → 환경변수 미설정

## 실제 작동 확인

메시지가 안 보여도 **테스트 결과는 성공**이므로 실제로는 잘 작동하고 있습니다!

확인 방법:
1. 웹앱에서 참가자 추가
2. 새로고침 후 데이터가 유지되는지 확인
3. 브라우저 콘솔(F12)에서 오류 확인
