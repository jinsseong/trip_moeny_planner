# 🔧 Chrome 확장 프로그램 오류 해결 가이드

## ❌ 오류 메시지

```
Failed to load resource: net::ERR_FAILED
chrome-extension://invalid/:1  Failed to load resource: net::ERR_FAILED
```

## 📋 원인

이 오류는 **Chrome 확장 프로그램**이 잘못된 리소스를 로드하려고 할 때 발생합니다. **여행 정산 웹앱 자체의 문제가 아닙니다.**

일반적인 원인:
- 설치된 Chrome 확장 프로그램이 페이지를 수정하려고 시도
- 확장 프로그램이 잘못된 리소스를 참조
- 확장 프로그램이 페이지 콘텐츠를 주입하려고 시도

## ✅ 해결 방법

### 방법 1: Chrome 확장 프로그램 비활성화 (가장 간단)

1. **Chrome 주소창에 입력**: `chrome://extensions/`
2. **확장 프로그램 목록 확인**
3. **의심스러운 확장 프로그램 비활성화**:
   - 광고 차단기
   - 번역 확장 프로그램
   - 개발자 도구 확장 프로그램
   - 기타 최근 설치한 확장 프로그램
4. **페이지 새로고침** (F5)

### 방법 2: 시크릿 모드에서 테스트

1. **Ctrl+Shift+N** (Windows) 또는 **Cmd+Shift+N** (Mac)
2. 시크릿 창에서 웹앱 접속
3. 확장 프로그램이 비활성화된 상태로 테스트

### 방법 3: 특정 확장 프로그램 확인

다음 확장 프로그램들이 문제를 일으킬 수 있습니다:
- AdBlock, uBlock Origin (광고 차단기)
- Google Translate (번역 확장 프로그램)
- React DevTools (개발자 도구)
- Redux DevTools (개발자 도구)
- 기타 개발자 도구 확장 프로그램

**확인 방법:**
1. 모든 확장 프로그램 비활성화
2. 하나씩 활성화하며 테스트
3. 문제를 일으키는 확장 프로그램 찾기

### 방법 4: Chrome 콘솔에서 필터링

1. **F12** (개발자 도구 열기)
2. **Console** 탭 클릭
3. 우측 상단 **필터 아이콘** 클릭
4. **"Hide network"** 또는 **"Hide extension errors"** 체크
5. 또는 필터에 `-chrome-extension` 입력

## 🧪 애플리케이션 정상 작동 확인

이 오류는 **애플리케이션 기능에 영향을 주지 않습니다**. 다음을 확인하세요:

### ✅ 정상 작동 확인 체크리스트

- [ ] 메인 페이지가 로드되는가?
- [ ] 참여자 추가/수정/삭제가 작동하는가?
- [ ] 지출 내역 추가/수정/삭제가 작동하는가?
- [ ] 정산 계산이 정상적으로 표시되는가?
- [ ] 관리자 페이지 로그인이 작동하는가?
- [ ] 데이터가 새로고침 후에도 유지되는가? (Supabase 연동 시)

### 🔍 실제 애플리케이션 오류 확인

**실제 애플리케이션 오류가 있다면:**
- Console에 빨간색 오류 메시지가 표시됨
- 애플리케이션 기능이 작동하지 않음
- 네트워크 탭에서 API 호출 실패

**Chrome 확장 프로그램 오류는:**
- 노란색 경고 메시지
- 애플리케이션 기능은 정상 작동
- `chrome-extension://invalid/` 형태의 URL

## 🛠️ 개발 환경에서 무시하기

개발 중에는 이 오류를 무시해도 됩니다. 하지만 콘솔이 깔끔하게 보이길 원한다면:

### Vite 개발 서버 설정 (선택사항)

`vite.config.js`에 다음을 추가할 수 있습니다 (하지만 일반적으로 필요 없음):

```javascript
export default defineConfig({
  // ... 기존 설정
  server: {
    // ... 기존 설정
    hmr: {
      overlay: false, // 오류 오버레이 비활성화 (선택사항)
    },
  },
});
```

## 📝 요약

- ✅ **이 오류는 무시해도 됩니다** - 애플리케이션 기능에 영향 없음
- ✅ **Chrome 확장 프로그램 문제** - 웹앱 코드 문제 아님
- ✅ **해결 방법**: 확장 프로그램 비활성화 또는 시크릿 모드 사용
- ✅ **실제 오류 확인**: Console에서 빨간색 오류 메시지 확인
